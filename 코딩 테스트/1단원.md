## STL의 컨테이너 
* 컨테이너 : 데이터를 저장하는 객체
  종류
  ---
  ## 벡터
  데이터를 순차적으로 저장하고 인덱스를 통해 특정 위치에 원소에 쉽게 접근
  헤더에 벡터 선언 #include <vector>
  vector<int> v3(4, 3) // 벡터크기 4 모든 원소는 3 // 3,3,3,3
  원소변경 : v3[0] = 2; // 2,3,3,3
  원소 삽입과 삭제 : v3.push_back(6); v3.pop_back(); // 맨뒤에 원소 6삽입, 맨뒤의 원소 삭제
  맨 앞에 삽입과 삭제 : v3.insert(v3.begin(), 1); v3.erase(v3.begin()); // 1,2,3,3,3;   2,3,3,3
  ---
  ## 셋
  중복을 허락하지 않고 저장된 데이터를 자동으로 정렬
  헤더에 셋 선언 #include <set>
  set<int> s2 = {3,1,3,2,5}; // 1,2,3,5출력
  원소찾기find()메서드 사용 // find()메서드 시간 복잡도는 빅오logN
